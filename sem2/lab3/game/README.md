# Crimsoland

Crimsoland - игра на `pygame`, где игрок отбивается от волн врагов разными видами оружия. Проект разделен на небольшие модули, чтобы основной файл `game.py` отвечал за запуск игрового процесса, а вспомогательная логика находилась в отдельных файлах.

## Запуск проекта

1. Установите `pygame`, если он еще не установлен.

```powershell
pip install pygame
```

2. Запустите игру из корня проекта.

```powershell
python game.py
```

## Структура проекта

```text
game.py          Главный игровой цикл и запуск проекта
audio.py         Фоновая музыка
controls.py      Обработка клавиатуры, мыши и выстрелов
player.py        Игрок: движение, здоровье, выбранное оружие
bullet.py        Пули: движение, столкновения, след от выстрела
evil.py          Враги: движение, атака, замедление, отбрасывание
effects.py       Визуальные эффекты попаданий и эффекты оружия
end_screen.py    Экран победы или поражения
loaders.py       Загрузка JSON-конфигов, картинок и анимаций
menu.py          Главное меню и сохранение лучшего счета
ui.py            Интерфейс игры: сердечки здоровья и баннер волны
waves.py         Создание волн и появление врагов
configs/         JSON-настройки игры
images/          Изображения персонажей, интерфейса, фонов и эффектов
sounds/          Музыка и звуки
```

## Главный сценарий игры

Основная логика находится в `game.py`.

Функция `main()`:

1. Инициализирует `pygame`.
2. Запускает фоновую музыку через `start_background_music()`.
3. Показывает главное меню.
4. Запускает игру через `run()`.
5. Если игрок нажимает повторный старт на финальном экране, запускает новую игру.

Функция `run()`:

1. Создает окно, шрифты, фон и кнопки.
2. Загружает конфиги оружия, врагов и волн.
3. Создает игрока, группы пуль и врагов.
4. Запускает первую волну.
5. В цикле обновляет игрока, врагов, пули, столкновения, эффекты и интерфейс.
6. Завершает игру победой, если все волны пройдены.
7. Завершает игру поражением, если здоровье игрока стало `0` или меньше.
8. Показывает экран результата через `show_end_screen()`.

## Конфиги

Настройки игры лежат в папке `configs`.

### `configs/weapon.json`

Файл описывает оружие.

Пример структуры оружия:

```json
{
  "star": {
    "image": "images/star.png",
    "size": [40, 40],
    "speed": 12,
    "damage": 2,
    "trail_width": 4,
    "effect": "direct",
    "animation_size": [90, 90],
    "animation_frame_delay": 3,
    "fallback_animation_image": "images/star.png"
  }
}
```

Основные поля:

- `image` - путь к картинке оружия.
- `size` - размер картинки оружия в пикселях.
- `speed` - скорость полета пули.
- `damage` - урон.
- `trail_width` - толщина следа от пули.
- `effect` - тип эффекта оружия.
- `animation_size` - размер анимации попадания.
- `animation_frame_delay` - задержка между кадрами анимации.
- `fallback_animation_image` - картинка, если кадры анимации не найдены.

Для бомбы дополнительно используются:

- `hit_radius` - радиус попадания самой пули.
- `explosion_radius` - радиус взрыва.
- `knockback` - сила отбрасывания врагов.
- `fallback_explosion_image` - запасная картинка взрыва.

Для алмаза дополнительно используются:

- `slow_factor` - во сколько раз замедляется враг.
- `slow_duration_ms` - длительность замедления в миллисекундах.

### `configs/enemies.json`

Файл описывает типы врагов.

Пример:

```json
{
  "basic": {
    "image": "images/man.png",
    "size": [110, 110],
    "speed": 0.9,
    "health": 8,
    "damage": 0.6,
    "stop_distance": 110,
    "attack_cooldown_ms": 850
  }
}
```

Основные поля:

- `image` - путь к картинке врага.
- `size` - размер врага.
- `speed` - скорость движения.
- `health` - здоровье.
- `damage` - урон по игроку.
- `stop_distance` - дистанция, на которой враг перестает подходить ближе.
- `attack_cooldown_ms` - задержка между атаками.

Поведение врагов задается в `evil.py`. Сейчас есть типы:

- `basic` - обычный враг.
- `fast` - быстрый враг.
- `tank` - медленный враг с большим здоровьем.
- `assassin` - быстро сближается и двигается агрессивнее.
- `ranger` - старается держать дистанцию.

### `configs/waves.json`

Файл описывает волны врагов.

Пример волны:

```json
{
  "enemies": [
    { "type": "basic", "count": 3 },
    { "type": "fast", "count": 1 }
  ]
}
```

Поля:

- `type` - тип врага из `configs/enemies.json`.
- `count` - количество врагов этого типа в волне.

Волны создаются по порядку. Когда все враги текущей волны уничтожены, запускается следующая. На последней волне появляется баннер `FINAL WAVE`.

### `configs/settings.json`

Файл хранит лучший счет:

```json
{
  "best_score": 0
}
```

Лучший счет обновляется на финальном экране через `menu.save_best_score()`.

## Ресурсы

### Изображения

Изображения лежат в папке `images`.

Основные изображения:

- `images/background.jpg` - фон главного меню.
- `images/field.jpg` - фон игрового поля.
- `images/game_over.jpg` - экран поражения.
- `images/win.jpg` - экран победы.
- `images/player.png` - игрок.
- `images/buttom_start.png` - кнопка старта.
- `images/exit.png` - кнопка выхода.
- `images/ui_hearts/` - кадры сердечек здоровья.

### Анимации эффектов

Кадры анимаций могут лежать в отдельных папках:

- `images/bomb_explosion/` - взрыв бомбы.
- `images/star_hit/` - попадание звездой.
- `images/diamond_hit/` - попадание алмазом.

Если папка с кадрами отсутствует или кадры не загрузились, используется запасная картинка из конфига оружия.

### Музыка

Фоновая музыка подключена в `audio.py`.

Чтобы добавить музыку, положите файл:

```text
sounds/background_music.mp3
```

Музыка запускается автоматически при старте проекта и играет по кругу.

Громкость задается в `audio.py`:

```python
BACKGROUND_MUSIC_VOLUME = 0.35
```

## Как добавить нового врага

1. Добавьте картинку врага в папку `images`.
2. Добавьте новый тип в `configs/enemies.json`.
3. Если нужно особое поведение, добавьте условие в метод `_get_target_velocity()` в `evil.py`.
4. Добавьте нового врага в нужные волны в `configs/waves.json`.

Пример:

```json
{
  "ghost": {
    "image": "images/ghost.png",
    "size": [120, 120],
    "speed": 1.3,
    "health": 9,
    "damage": 1.0,
    "stop_distance": 100,
    "attack_cooldown_ms": 700
  }
}
```

## Как добавить новое оружие

1. Добавьте картинку оружия в папку `images`.
2. Добавьте новый объект в `configs/weapon.json`.
3. Добавьте переключение на оружие в `controls.py`.
4. Если у оружия новый эффект, добавьте обработку в `effects.py` и место вызова в `game.py`.

Для простого оружия можно ориентироваться на `star`, для взрывного - на `bomb`, для замедляющего - на `diamond`.

## Где что менять

- Хотите поменять сложность волн - редактируйте `configs/waves.json`.
- Хотите поменять здоровье, скорость или урон врагов - редактируйте `configs/enemies.json`.
- Хотите поменять урон, скорость или размер оружия - редактируйте `configs/weapon.json`.
- Хотите поменять музыку - замените `sounds/background_music.mp3`.
- Хотите поменять управление - редактируйте `controls.py`.
- Хотите поменять движение врагов - редактируйте `evil.py`.
- Хотите поменять интерфейс здоровья или баннер волны - редактируйте `ui.py`.
